React.js هي مكتبة JavaScript قوية لبناء واجهات المستخدم التفاعلية. في هذا المقال، سنتعلم الأساسيات والمفاهيم المهمة.

JavaScript 
import React from 'react';

function App() {
  const [count, setCount] = React.useState(0);

  return (
    <div>
      <h1>Counter: {count}</h1>
      <button onClick={() => setCount(count + 1)}>
        Increment
      </button>
    </div>
  );
}

export default App;

هذا المثال يوضح كيفية استخدام useState Hook لإدارة الحالة في مكون React وظيفي.
